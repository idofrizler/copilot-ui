# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Cooper, please report it responsibly by emailing **ido.frizler@gmail.com**.

**Please do not open a public GitHub issue for security vulnerabilities.**

When reporting, include as much of the following as you can:

- A description of the vulnerability
- Steps to reproduce the issue
- The potential impact
- Any suggested fix (if applicable)

You can expect an initial response within 72 hours. Once the issue is confirmed, a fix will be prioritized and released as soon as possible, along with credit to the reporter (unless anonymity is preferred).
